🔄 الترحيل إلى Supabase

الحالة الحالية

الخادم يعمل حالياً بـ SQLite (ملف محلي). يمكنك الترحيل إلى Supabase PostgreSQL للحصول على قاعدة بيانات سحابية.

Database: SQLite (محلي)

خطوات الترحيل

  1. افتح Supabase Dashboard
  2. اذهب إلى مشروعك → SQL Editor
  3. الصق محتوى Schema أدناه
  4. اضغط Run
  5. أضف SUPABASE_URL و SUPABASE_SERVICE_KEY في gateway/.env
  6. أعد تشغيل الخادم
